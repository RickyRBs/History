# Cíqì Chronicles - Standalone Version

This is a simplified, standalone version of the Cíqì Chronicles Timeline application.

## How to Deploy

You do **not** need to run any build commands (`npm install`, `vite`, etc.) for this version.

1.  **Upload `index.html`** to your GitHub Repository.
2.  Go to your Repository Settings > **Pages**.
3.  Select the **main** branch as the source and click **Save**.
4.  Your site will be live!

## Editing the Code

All the logic (React components, data, styles) is contained within `index.html`. You can edit this file directly in any text editor.
